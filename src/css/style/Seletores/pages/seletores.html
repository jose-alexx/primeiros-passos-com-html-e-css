No CSS nós possuimos os seletores por ID

  • Seletor por tipo: Busca elementos por tag HTML especifica
  • Seletor por ID (#): Busta elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor universal (*): Seleciona todos os elementos HTML
  • Seletor de atributo ([atrib]):Seleciona elementos que possuiem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico.
